What is Winged Monkey?

Winged Monkey is an application that provides an elegant and intuitive user interface for self-service consumers of cloud resources. It offers simplified interactions for non-administrative users, with some specific goals:

  • Browse a list of applications that the user can launch
  • Launch an application with the minimum necessary steps
  • See the status of applications which the user has launched
  • Terminate an application that the user has launched
